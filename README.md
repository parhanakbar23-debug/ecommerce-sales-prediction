# 🛒 E-Commerce Sales Prediction\

Nama : parhan Akbar
Nim  : 301240038


Sistem prediksi penjualan E-Commerce menggunakan Machine Learning dan Deep Learning berbasis Python Flask.

---

# 📌 Deskripsi Project

Project ini dibuat untuk memenuhi Ujian Tengah Semester (UTS) mata kuliah kecerdasan buatan / machine learning.

Sistem mampu:
- melakukan analisis data penjualan
- preprocessing data
- prediksi total pembayaran
- perbandingan performa model AI
- visualisasi dashboard interaktif

---

# 📊 Dataset

Dataset berisi:
- transaksi penjualan e-commerce
- metode pembayaran
- kategori produk
- ongkir
- diskon
- status pesanan
- waktu transaksi

Jumlah data:
- 20.848 transaksi

---

# 🧠 Model yang Digunakan

## Machine Learning
- Linear Regression
- K-Means Clustering

## Deep Learning
- Artificial Neural Network (ANN)
- LSTM
- Backpropagation Neural Network
